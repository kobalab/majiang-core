# majiang-core

麻雀基本ライブラリ

## インストール
```sh
$ npm -i @kobalab/majiang-core
```

## 使用法
```javascript
const Majiang = require('@kobalab/majiang-core');
```

## 提供機能
| クラス名            | 機能                                 |
|:--------------------|:-------------------------------------|
| ``Majiang.Shoupai`` | 手牌を表現するクラス                 |
| ``Majiang.Shan``    | 牌山を表現するクラス                 |
| ``Majiang.He``      | 捨て牌を表現するクラス               |
| ``Majiang.Util``    | シャンテン数計算、和了点計算ルーチン |
| ``Majiang.Game``    | 局進行を実現するクラス               |
